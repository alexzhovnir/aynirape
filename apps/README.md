# apps/

Deployable applications.

- **`backend/`** — Medusa v2 commerce backend + admin. Scaffold with
  `create-medusa-app` (see [../SETUP.md](../SETUP.md)).
- **`storefront/`** — Astro storefront. Scaffold with `create astro`.

Both are created by their official generators, then wired into the pnpm workspace.
