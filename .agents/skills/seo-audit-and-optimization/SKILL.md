# SEO Audit & Optimization Best Practices

Use this skill to enforce and audit SEO implementations.

## Audit Checklist
1. **Dynamic Titles & Metadata**: Verify every page has a unique `<title>` and `<meta name="description">` suited for search intent.
2. **Canonical Links**: Ensure `<link rel="canonical" href="...">` matches the exact routing pattern.
3. **Structured Schema JSON-LD**: Ensure `@type: Product`, `@type: Organization`, and `@type: Article` schema markup is placed where appropriate with complete specifications.
4. **Semantics**: Ensure only one `<h1>` per page is rendered and heading elements form a strict descending hierarchical order.
5. **Interactive Element Accessibility**: Ensure all critical interactive features contain clean `id` properties, `alt` labels for images, and ARIA attributes for accessibility tools.
