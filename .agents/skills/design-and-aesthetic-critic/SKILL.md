# Design & Aesthetic Critique and Optimization

Use this skill to ensure high-end, premium visual aesthetics matching state-of-the-art standards.

## Audit Checklist
1. **Color Harmony**: Check if plain primary colors (simple red, blue, green) are used. Replace them with curated, high-end palettes (e.g., deep forest green `#1e2d24`, warm amber `#eab308`, slate gray `#9db0ba`).
2. **Typography**: Ensure default system fonts are overridden with premium typography (Outfit, Inter, Playfair Display) loaded via Google Fonts or Tailwind configurations.
3. **Geometry & Spacing**: Elements (cards, headers, images) must use unified rounded corners (`rounded-3xl` / 24px-32px radius) and consistent padding/gap spacing.
4. **Interactive States**: Every link, button, and action must have clean transition animations (`transition-all duration-300`) and clear hover states.
5. **Layout & Glassmorphism**: Use backdrop filters (`backdrop-blur-md bg-white/80`) to achieve clean layered designs.
